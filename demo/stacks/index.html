SmartField — Multi-Stack Demos

Same component, 6 server languages. Each demo encrypts in the browser and decrypts on its own server.

PORT 3333

Node.js

Express server. The original SmartField SDK. Zero dependencies.

 PORT 5555

Python

Flask server. Uses cryptography library for RSA + AES-GCM.

 PORT 6666

Java

Lightweight HTTP server. javax.crypto for all encryption.

 PORT 7777

Go

net/http server. Standard crypto library. Zero dependencies.

 PORT 8888

PHP

Built-in PHP server. openssl extension for RSA + AES-GCM.

 PORT 9999

Ruby

WEBrick server. OpenSSL library for encryption.

Each server generates its own RSA-2048 keys. The component encrypts with AES-256-GCM. Only that server can decrypt.
